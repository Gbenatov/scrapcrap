# 📊 Data Scraper & Exporter

אפליקציית Streamlit אינטראקטיבית להורדה, עיבוד וייצוא נתונים מאתרים.

## ✨ תכונות

### 🌐 שלוש אפשרויות קלט:

1. **הזן URL** - הזן כתובת אתר וחלץ טבלאות בעצמך
2. **העלה קובץ** - תמיכה ב-CSV, Excel (XLS, XLSX)
3. **דוגמה** - הצג נתוני דוגמה מובנים

### 🔧 אפשרויות עיבוד:

- ✅ בחר/בטל עמודות לייצוא
- ✅ סנן מספר שורות
- ✅ תצוגה מקדימה של הנתונים
- ✅ סטטיסטיקה ודיאגרמות

### 💾 פורמטי ייצוא:

- 📄 CSV
- 📊 JSON
- 📈 Excel (XLSX)

## 🚀 הפעלה

### באתר (Streamlit Cloud):
https://your-app-url.streamlit.app

### בלוקלי:
```bash
streamlit run streamlit_app.py
```

## 💡 דוגמאות שימוש

### 1. הזנת URL:
```
כתובת האתר: https://example.com/data
→ טען דף
→ בחר עמודות
→ הורד CSV/JSON/Excel
```

### 2. העלאת קובץ:
```
בחר: data.csv
→ בחר עמודות
→ סנן שורות
→ הורד בפורמט חדש
```

### 3. דוגמה:
```
לחץ "דוגמה"
→ ראה נתוני דוגמה
→ נסה את הפונקציונליות
```

## 🔍 בעיות נפוצות

**שגיאה בטעינת דף:**
- בדוק שה-URL תקין
- בדוק שיש חיבור אינטרנט
- בדוק שהדף מכיל טבלאות

**לא מוצאים אפשרויות Export:**
- בחר לפחות עמודה אחת
- ודא שיש נתונים בטבלה

## 📝 הערות

- התמיכה בטבלאות דינמיות (JavaScript) מוגבלת
- גודל קובץ מקסימלי: ~100 MB
- תמיכה בעברית מלאה
